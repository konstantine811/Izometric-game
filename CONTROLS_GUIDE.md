# 🎮 Керування - Beergame

## 🕹️ Основне керування

### Рух
```
     W / ↑
      |
A / ← + → / D
      |
    S / ↓
```

| Клавіша | Дія |
|---------|-----|
| **W / ↑** | Рух на північ |
| **A / ←** | Рух на захід |
| **S / ↓** | Рух на південь |
| **D / →** | Рух на схід |
| **Shift + WASD** | Біг (2× швидше) |

**Діагональний рух:** W+D, W+A, S+D, S+A

---

### Миша
- **Лівий клік** - рух до точки (A* pathfinding)
- Автоматичний обхід перешкод

---

## ⚔️ Action Анімації (Warrior)

| Клавіша | Дія | Тривалість |
|---------|-----|------------|
| **Space** | Flip (стрибок) | ~0.5s |
| **R** | Roll (перекат) | ~0.7s |
| **Q** | Melee (меч) | ~0.7s |
| **E** | Cast (магія) | ~0.7s |
| **K** | Kick (удар ногою) | ~0.7s |
| **T** | Turn (поворот 180°) | ~0.7s |

**Особливості:**
- ✅ Виконуються ОДРАЗУ (зупиняють рух)
- ✅ Неможливо перервати
- ✅ Автоматичне повернення до idle

---

## 🎨 Зміна персонажів

| Клавіша | Дія |
|---------|-----|
| **C** | Переключити персонажа |

**Персонажі:**
1. **Hero** - класичний (8 кадрів walk)
2. **Cyberpunk Marsian** - кіберпанк (6 кадрів walk)
3. **Warrior** - воїн ⚔️ (15 кадрів walk/run/actions)

## 🗺️ Ізометрична проекція

```
       N (Північ)
        ↑
   NW ↖ │ ↗ NE
        │
W ← ────┼──── → E
        │
   SW ↙ │ ↘ SE
        ↓
       S (Південь)
```

**8 напрямків руху** - анімація автоматично підбирається!

---

## 🎯 Поради

### Ефективний рух
- **Клавіатура** - точний контроль
- **Миша** - довгі дистанції
- **Shift** - для швидкого пересування

### Діагональ швидше
```
Швидше:  W+D (діагональ)
Повільніше: W → D → W → D (зигзаг)
```

### Action комбо (Warrior)
```
1. Підійти до ворога (WASD)
2. Melee атака (Q)
3. Відстрибнути (Space назад)
4. Cast магію (E)
```

## 🎬 Анімації

### Warrior (15 кадрів)
- **Walk** - ходьба (30 FPS)
- **Idle** - стояння (18 FPS)
- **Run** - біг (30 FPS, Shift)
- **6 action анімацій** (Space, R, Q, E, K, T)

### Cyberpunk (6 кадрів)
- **Walk** - ходьба (10 FPS)
- **Idle** - стояння (8 FPS)

### Hero (8 кадрів)
- **Walk** - ходьба (13 FPS)
- **Idle** - стояння (8 FPS)

## 🐛 Обмеження

### Що НЕ можна:
❌ Перервати action анімацію  
❌ Проходити крізь стіни  
❌ Рухатися за межі карти  

### Що можна:
✅ Міняти напрямок під час руху  
✅ Утримувати клавіші для безперервного руху  
✅ Міняти персонажа в будь-який момент  
✅ Action анімації виконуються одразу
